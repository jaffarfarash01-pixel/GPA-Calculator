# 🎓 GPA Calculator

A simple and user-friendly web application to calculate **SGPA (Semester GPA)** and **CGPA (Cumulative GPA)**.

---

## 🚀 Features

- ➕ Add subjects with credits and grades
- 🗑️ Delete subjects
- 📊 Calculate SGPA instantly
- 🔄 Separate pages for SGPA and CGPA
- 🎨 Clean and responsive UI
- ⚡ Fast and lightweight (no frameworks)

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

---


---

## ▶️ How to Run

1. Download or clone the repository
2. Open `index.html` in your browser
3. Use the buttons to navigate:
   - SGPA Calculator
   - CGPA Calculator
